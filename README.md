# Decoding Emotions Through Sentiment Analysis of Social Media Conversations

## 🧠 About the Project

This project is part of the **Naan Mudhalvan Phase 2** initiative. It focuses on analyzing social media text to decode human emotions using sentiment analysis.

## 📁 Files Included

- `dataset.csv` – Sample social media comments with labeled sentiments
- `source_code.py` – Python code for text processing and sentiment classification
- `Output (Google Colab Implementation).pdf` – PDF showing sample results from the model

## 💡 Features

- Text Preprocessing and Vectorization
- Machine Learning Classification using Naive Bayes
- Real-time Sentiment Prediction on Custom Inputs
- Easy-to-follow implementation

## 🛠️ Tools Used

- Python
- Scikit-learn
- Google Colab
- Pandas & CountVectorizer
